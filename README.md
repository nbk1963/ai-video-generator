This repo is a frontend template (HTML/CSS/JS) for an AI text-to-video generator. It contains:
- Responsive UI
- Modal generator form
- Prompt cards
- Theme toggle (dark/light)

**Important**: This is a front-end demo. To enable real video generation you must implement a backend API that accepts a prompt and returns a generated video (or a job ID). See `src/app.js` for the demo flow.

To run locally:
1. Clone repo
2. Open `index.html` in a browser
3. Replace `assets/sample-video.mp4` with your own sample or production output.
